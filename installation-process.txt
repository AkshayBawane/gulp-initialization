1. install node
2. npm install --global gulp-cli or npm install --global gulp
3. npm install --save-dev gulp
4. npm install --save-dev babel-core
6. npm install --no-optional
6. npm install babel-register --save-dev
6. npm rm -rf node_modules
6. npm cache verify
5. npm install

Change dependencies of boootstrap version to latest,

Help:
https://gulpjs.org/getting-started
https://github.com/jakearchibald/wittr/issues/20
https://github.com/yeoman/generator-webapp/issues/356

Uninstall:
1. npm uninstall gulp --save-dev
2. npm uninstall gulp -g
